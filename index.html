An Experimental Study on Brain Tumor Detection using Deep Learning Techniques (Presented At: IIT-Mandi)

15th International Conference on Computing Communication and Networking Technologies (ICCCNT) |IIT-Mandi|

In this study, advanced deep learning approaches, Inception V3, GoogLeNet, and VGG-19, were applied to MRI based brain tumor detection. We implemented decision level fusion to combine multiple model outputs, significantly boosting classification precision. The models achieved high accuracy (>98%) for tumor identification, underscoring the power of convolutional neural networks and transfer learning in medical image analysis and healthcare diagnostics.

IEEE Publication

A Comparative Analysis of Object Detection Models for Assistive Navigation in Dynamic Environments (Presented At: Tokyo, Japan)

11th IEEE International Conference on Applied System Innovation 2025 (IEEE-ICASI 2025) |Tokyo, Japan|

This research evaluates leading object detection models including YOLOv11, DETR, CenterNet, and Faster R-CNN using a custom dataset to improve assistive navigation for visually impaired individuals. By benchmarking these deep learning architectures, our study demonstrates how model selection impacts real-time inference speed and accuracy. We highlight YOLOv11‚Äôs suitability for edge-device deployment, emphasizing the importance of optimizing ML models for accessibility and dynamic real-world challenges

IEEE Publication

Assistive Navigation for the Visually Impaired Users Using YOLOv11 with Depth Enhancement and Object Detection

This research develops a three-phase assistive navigation system for the visually impaired, benchmarking top object detection models and selecting YOLOv11 for real-time edge deployment. It integrates YOLOv11 with monocular depth estimation (MiDaS) and rule-based risk classification to provide multimodal hazard alerts. The system is optimized through model compression and pruning for efficient, personalized use in dynamic environments.
